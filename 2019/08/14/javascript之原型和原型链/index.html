Prototype

函数的 prototype 属性指向一个对象，这个对象就是 调用这个构造函数创建的实例的原型，也就是 person1 和 person2 的原型

__proto__

每个对象都有一个 __proto__ 属性，指向它的原型，也就是其构造函数的prototype属性

constructor
